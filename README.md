# 설명

이 사이트는 Firebase, React를 연습하기 위해서 만든 사이트이다.

Firebase의 realtime database를 사용하여 입력하는 즉시 실시간으로 저장가능하다.

또한 Firebase에서 제공하는 로그인 기능을 활용하여 구글, 깃허브로 로그인이 가능하다.
